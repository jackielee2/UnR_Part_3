		-	SpriteSize를

		-----------------

			----------
			-	개요..
			----------

				-	위에서 설정한 정보를
					바탕으로 프로그램 작성..

				-	블루프린트( 이하 BP )를
					이용하는 방식이므로
					BP에 대한 지식 필요..
					-	필요한 처리를 담당하는
						노드를 작성하여
						서로 연결한 뒤
						순서대로 실행하는 방식..


			----------
			-	절차..
			----------

				1)	[ Map Get ]노드 에
					핀 추가..

					방법1)	[ + ]
							> [ 기존 파라미터 추가 ]
							-	addValue 선택..

					방법2)	[ 파라미터 창 ]
							> [ 모듈 입력 ]
							> [ addValue ]를
							[ Map Get ]노드에 드래그..


				2)	추가 노드 생성..

					-	두 값을 더하는 노드 추가..

						1)	[ 마우스 우클릭 ]
							> " + " 검색..
							-	[ 추가 ] 노드 생성..

						2)	[ Map Get ]
							> [ SpriteSize ], [ addValue ]를
							[ 추가 ] > [ A ], [ B ]에
							각각 연결..


				3)	Vector2D 만들기
					노드 생성..

					-	[ Make Vector 2D ]

					-	스프라이트의 크기를
						설정하기 위해 필요..
						

				4)	[ Map Set ]노드 에
					핀 추가..
					-	[ SpriteSize ] 연결..
					-	스프라이트의 크기 설정..


				5)	노드 연결..

					1)	[ 추가 노드 ]
						> [ 결과 핀 ]
						
						--->	[ Make Vector 2D ]
								> [ X ]
						--->	[ Make Vector 2D ]
								> [ Y ]

					2)	[ Make Vector 2D ]
						> [ Output1 ]

						--->	[ Map Set ]
								> [ SpriteSize ]


				6)	모듈 적용..

					-	그래프 좌측 상단의
						[ 적용 ] 버튼 클릭..
